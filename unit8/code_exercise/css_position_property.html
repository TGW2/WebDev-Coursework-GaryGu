• Static positioning

    With static positioning, the elements are positioned according to the normal flow of your HTML document. The elements are not affected by the top, bottom, left, or right properties.

    This is a static positioned element.

  • Relative positioning

    With relative positioning, an element is positioned relative to its normal or default position. You use the top, bottom, left, and right properties to position the element away from its normal position.

    This is a relatively positioned element.

  • Absolute positioning

    With absolute positioning, an element is positioned relative to its closest positioned ancestor (compare this with fixed positioning.)

    This is an absolutely positioned element.

  • Fixed positioning

    With fixed positioning, an element will always stay in the same place, even when the user scrolls along the web page.

    This is an absolutely positioned element.
